# Spec Driven Development

## Examples

1. Read .specs/002-storage-layer/requirements.md. Draft a design.md in the same folder. It outlines the overall architecture and design for this feature.
2. Read .specs/002-storage-layer/design.md. Draft a tasks.md in the same folder. It outlines the step-by-step tasks to implement as individual checkboxes which the agent will check each one when finished.
3. Read .specs/002-storage-layer/tasks.md. Draft a tests.md in the same folder. It outlines all the happy path and edge case tests that need to be implemented.
4. Read .specs/002-storage-layer/*.md and implement the feature. Follow the task checklist in tasks.md. Update the checkboxes as you complete each item. Run the tests before marking complete.
